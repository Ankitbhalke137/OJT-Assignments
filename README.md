# OJT-Assignments
Hands on Assignments Advance JS @ *

<b>1. Click Counter App</b>
Problem Statement: Create a web page with buttons to increment, decrement, and reset a counter. Display the current count dynamically.
Problem: Build a counter interface that updates based on user interaction.
UI Requirements:
Display area for count
Three buttons: Increment, Decrement, Reset
Constraints:
Count should not go below 0
Use DOM to update value dynamically
Use click events only

2. Toggle Light/Dark Mode
Problem Statement: Design a webpage with a button that switches between light and dark themes using CSS classes.
Problem: Switch website theme dynamically.
UI Requirements:
Toggle button
Visible color changes (background, text)
Constraints:
Use classList.toggle()
Maintain state during session (optional: localStorage)

3. Show/Hide Password
Problem Statement: Build a password field with a checkbox or icon to toggle visibility of the
entered password.
Problem: Toggle password visibility.
UI Requirements:
Password input
Checkbox or icon
Constraints:
Switch input type attribute
No page reload

4. Alert Button System
Problem Statement: Create buttons that trigger different alert messages when clicked.
Problem: Trigger different alerts.
UI Requirements:
Multiple buttons (Success, Warning, Error)
Constraints:
Use event listeners
Messages must differ per button

5. Random Background Color
Problem Statement: Generate and apply a random background color when a button is clicked.
Problem: Change background color randomly.
UI Requirements:
Button + display hex code
Constraints:
Generate valid hex color
Update DOM style dynamically

6. Button Color Changer
Problem Statement: Change the color of a button each time it is clicked.
Problem: Change button color on each click.
Constraints:
Cycle through at least 3 predefined colors

7. Digital Clock
Problem Statement: Display a real-time digital clock that updates every second.
Problem: Display real-time clock.
UI Requirements:
HH:MM:SS format
Constraints:
Use setInterval()
Update every second

8. Greeting Based on Time
Problem Statement: Show a greeting message (Good Morning/Afternoon/Evening) based on the current time.
Problem: Show greeting depending on hour.
Constraints:
Use system time
At least 3 conditions (morning/afternoon/evening)

9. Image Toggle
Problem Statement: Add a button to show or hide an image dynamically.
Problem: Show/hide image.
Constraints:
Use style.display or class toggle
